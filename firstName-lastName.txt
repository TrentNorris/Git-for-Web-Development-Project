1. What is Semantic HTML? Semantic HTML is the use of semantic tags for improved readability and accessibility. 
2. What is HTML used for? HTML is a markup language for documents designed to be displayed in a web browser.
3. What is an attribute and where do we put it? An attribute is some added information, sometimes neccesary for the tag to function. It goes in the opening tag (<img src='website.png'>)
4. What is the h1 tag used for? How many times should I use it on a page? The h1 tag is the largest header size, it can be used as many timea as needed on a page.

7. What is an id? - A selector that is given to a single element.
8. What elements can I add an id to? - Any, but only one element.
9. How many times can I use the same id on a page? - Once. 
10. What is a class? - A class is a selector that can be used multiple times on a page to group elements together.
11. What elements can I add a class to? - All.
12. How many times can I use the same class on a page? - As many as needed.
13. How do I get my link to open in a new tab? -  add target="_blank" to your tag
14. What is the alt attribute used for? - It's an alternate way of displaying information(text), used for images, etc.
15. How do I reference an id? - place a # before the id name (example: #my-id)
16. What is the difference between a section and a div - A section is a semantic tag that is more accessible and SEO friendly. A div is a non-semantic grouping tag.
17. What is CSS used for? - CSS is used to style documents.
18. How to we select an element? Example, every h2 on the page - ( h2 {style here} )
19. What is the difference between a class and an id? Give me an example of when I might use each one - A class can be used on an infinite number of elements, where an id is used to select a specific element. I might make a class called 'center' and put "text-align: center;". Now I can add my center class to any element I need centered. I have a calendar that I need to select, but there's 3 calendars on the page and I just want to easily select it, so I'll add an id of 'calendar' to it. Now I can do #calendar { add my style here }.
